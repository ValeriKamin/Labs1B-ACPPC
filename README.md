Практична робота №1В
Розв’язання задачі лінійного програмування методом модифікованих жорданових виключень

Програма реалізована мовою C# та призначена для розв’язання задач лінійного програмування симплекс-методом.

Можливості програми
введення задачі вручну;
підтримка задач:
max
min
підтримка обмежень:
<=
>=
=
побудова симплекс-таблиці;
пошук опорного розв’язку;
пошук оптимального розв’язку;
покроковий протокол обчислень.
Технології
C#
.NET Console Application
Visual Studio
